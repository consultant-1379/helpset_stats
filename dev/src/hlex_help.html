Help
Using the Online Help
Online help is shown in a browser window with three separate frames. The toolbar frame on the top contains buttons with functions described below. On the left, there is a navigation frame with the help topics. The help topics can be shown as a tree structure or as an alphabetically sorted index list. The content frame to the right shows the actual help information for chosen topics.

Resizing the Browser Frames
Position the cursor on the vertical border (the cursor shape changes). Click and drag the frame border to a new position.

Using the Toolbar Buttons
Search help contents for a matching string.
Enter a search string in the search input field and press the Enter key or click the Search button to start a full text search. Search is always performed on all topics starting from the first one. All search hits found under a topic are highlighted with orange background color. The search is not case sensitive. Repeat the search to view the next topic that contains search hits.
Go to the next topic that contains search hits.
Go back to the previously shown topic.
Go forward to the next topic.
Switch to tree view.
Help topics are shown in a tree structure in the navigation frame. Click the (+) and (-) icons to expand and collapse the tree. Click a topic title or icon to view the topic content.
A selected topic can contain sub topics shown together in the content frame. Thus, to view all help topics, select the root topic in the tree. This is useful, for instance, to simultaneously view more than a single help topic or to make a printout of selected parts, or of the complete help information.
Switch to index view.
The help index consists of an alphabetically ordered list of topic titles shown in the navigation frame. Click a topic title to show the desired topic content.
Show help set in PDF.
Open the help set as a complete PDF document for high quality printouts, if this button is available.
Print the help topic content currently shown in this frame.
Show this help page.
